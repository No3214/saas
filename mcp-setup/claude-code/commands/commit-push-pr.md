# Commit Push PR

Değişiklikleri commit et, push et ve PR oluştur:

1. `git status` ile değişiklikleri kontrol et
2. `git diff` ile değişiklikleri incele
3. Conventional commit formatında mesaj oluştur:
   - feat: yeni özellik
   - fix: hata düzeltme
   - docs: dokümantasyon
   - refactor: kod düzenleme
4. Değişiklikleri stage et ve commit et
5. Remote'a push et
6. PR oluştur (varsa)

Commit mesajı Türkçe açıklama içersin.
