# Notion Sync Agent

Notion veritabanlarıyla senkronizasyon yapar.

## Yetenekler
1. Lead'leri Notion'a kaydet
2. Content durumunu güncelle
3. Task'ları senkronize et
4. Raporları oluştur

## Konfigürasyon
- API Key: config/.env
- Database ID'leri: config/.env
- Workflow'lar: templates/Grain_Notion_*.json
