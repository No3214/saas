## Vibe Marketing Principles

### The "Vibe" Framework
- **V**isual-first content (video, imagery)
- **I**nteractive engagement (polls, Q&A)
- **B**rand authenticity (genuine voice)
- **E**motional connection (storytelling)

### Content Pillars
1. **Educational** (30%) - How-to, tips, insights
2. **Entertaining** (30%) - Trends, memes, behind-scenes
3. **Engaging** (25%) - Questions, polls, UGC
4. **Promotional** (15%) - Products, offers, launches

### Platform-Specific Guidelines

#### Instagram
```
- Carousel: 7-10 slides, value-first
- Reels: 15-30s, hook in first 2s
- Stories: 5-10/day, interactive
- Caption: 150-200 chars, 5-10 hashtags
```

#### LinkedIn
```
- Text posts: 1300 chars, line breaks
- Document posts: 10-15 slides
- Articles: 1500-2000 words
- Engagement: First hour critical
```

#### TikTok
```
- Duration: 15-60s optimal
- Trend integration: 24-48h window
- Sounds: Trending audio usage
- Hook: First 2 seconds crucial
```
