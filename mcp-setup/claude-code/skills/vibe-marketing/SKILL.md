# Vibe Marketing Skill

Modern video-first social media marketing metodolojisi.

## Vibe Marketing Nedir?

Vibe Marketing, AI destekli, video-odakli, trend-takipcisi bir pazarlama yaklasimidir:

- **AI-Native**: Icerik uretiminde AI kullanimi
- **Video-First**: Reels, TikTok, Shorts onceligi
- **Trend-Driven**: Viral trendlere hizli adaptasyon
- **Data-Informed**: Performance verisiyle optimize

## Reels Script Formulu

### Hook-Content-CTA Yapisi

```
[HOOK - 0-2 saniye]
Dikkat ceken acilis

[CONTENT - 3-25 saniye]
Deger veren icerik

[CTA - Son 3-5 saniye]
Aksiyon cagrisi
```

### Hook Tipleri
```
1. Soru: "Neden hala X yapmiyorsun?"
2. Shock: "Bu hatayi herkes yapiyor!"
3. Promise: "3 adimda Y nasil yapilir"
4. Relatable: "Bu his herkeste var..."
5. Controversy: "Populer ama yanlis!"
```

### CTA Examples
```
- "Takip et, daha fazlasi icin"
- "Kaydet, lazim olacak"
- "Yorumlarda soyle..."
- "Link bio'da"
- "Paylas, arkadasin da ogrensin"
```

## Video Format Templates

### 15-Saniye Quick Tip
```
[0-2s] Hook sorusu
[3-12s] Tek net cevap
[13-15s] Follow CTA
```

### 30-Saniye Tutorial
```
[0-3s] Problem statement
[4-7s] Adim 1
[8-14s] Adim 2
[15-22s] Adim 3
[23-27s] Sonuc gosterimi
[28-30s] Save/Follow CTA
```

### 60-Saniye Story
```
[0-5s] Dikkat cekici hook
[6-15s] Problem/durum aciklamasi
[16-35s] Cozum/yolculuk
[36-50s] Sonuc/transformation
[51-60s] Lesson + CTA
```

## Viral Mechanics

### Psychological Triggers
```
1. Curiosity Gap: "Tahmin edemezsiniz..."

3. FOMO: "Son 24 saat!"
4. Identity: "Girisimciler bunu bilir"
```

## Hashtag Strategy

### Hashtag Mix Formula
```
- 1-2 Niche specific (50K-200K)
- 2-3 Industry related (200K-1M)
- 1-2 Trending (1M+)
- 1 Brand hashtag
- Total: 5-8 hashtag optimal
```
